# Signup Form

## About

This is a project from The Odin Project where they provide a layout of a signup form and you must create it from scratch.

They provide you some sources to get a font and background picture however you can also use one of your choosing.

This is **not** needing to be mobile friendly, so the layout should look good only on laptops or bigger screens in general.
